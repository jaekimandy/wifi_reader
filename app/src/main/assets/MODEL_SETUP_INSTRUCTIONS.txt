PaddleOCR Model Setup Instructions
========================================

To get working OCR models:

Option 1 - Use pre-converted TensorFlow Lite models:
1. Visit: https://github.com/tensorflow/examples/tree/master/lite
2. Download text detection/recognition .tflite models
3. Rename them to:
   - paddleocr_detection.tflite
   - paddleocr_recognition.tflite

Option 2 - Convert PaddleOCR models:
1. Install: pip install paddle2onnx onnx tf2onnx
2. Download PaddleOCR models from above URLs
3. Convert: paddle2onnx -> onnx2tf -> .tflite

Option 3 - Use the enhanced mock mode:
The PaddleOCREngine will run in enhanced mock mode
and generate realistic WiFi router label text for testing.
